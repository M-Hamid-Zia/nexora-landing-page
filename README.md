# NEXORA Landing Page

A clean, responsive landing page built entirely with vanilla HTML and CSS. This project was created to strengthen fundamental web development skills, focusing on modern layout techniques without relying on frameworks.

## Features

- Modern Layout Techniques: Utilizes both Flexbox and CSS Grid for structured, maintainable layouts

- Fully Responsive Design: Adapts seamlessly from mobile to desktop screens

- Semantic HTML: Properly structured with meaningful elements for accessibility and SEO

- Smooth Interactions: Subtle hover effects and transitions for enhanced user experience

- Clean Visual Design: Careful attention to spacing, typography, and color harmony

## What I Learned & Practiced

- Layout & Structure

  - Creating complex layouts using pure CSS Flexbox and Grid
  - Implementing mobile-first responsive design principles
  - Structuring projects without CSS frameworks for better understanding of core concepts

- CSS Skills

  - Advanced spacing and alignment control with margin, padding, and positioning
  - Creating smooth hover effects and transitions for interactive elements
  - Using CSS custom properties for consistent theming
  - Implementing responsive typography with relative units

- Development Practices
  - Writing clean, semantic HTML for accessibility and maintainability
  - Organizing CSS with logical grouping and comments
  - Browser debugging and testing across different devices
  - Version control best practices with Git and GitHub
